# Dr. Mahmoud Omran Dental Clinic Landing Page 🦷✨

A premium, high-fidelity, and fully responsive landing page for **Dr. Mahmoud Omran's Specialized Dental Clinic** in Minia, Egypt. Built with modern web technologies to provide a luxury user experience and high conversion for dental services.

## 🚀 Live Demo
**GitHub Repository**: [Dental-Landing-page](https://github.com/jodx19/Dental-Landing-page.git)

---

## ✨ Features

- **💎 Luxury Design**: Modern golden-themed aesthetic with a premium clinical feel.
- **📱 Fully Responsive**: Optimized for all devices—mobiles, tablets, and desktops.
- **🌐 Multi-language Support**: Seamless toggle between Arabic and English (LTR/RTL support).
- **🎨 High Contrast & Accessibility**: Optimized for readability with a strict 4.5:1 contrast ratio.
- **⚡ Performance**: Built with Vite and React for ultra-fast loading times.
- **💬 Direct Booking**: Integrated WhatsApp booking system for easy patient communication.
- **✨ Smooth Animations**: Powered by Framer Motion for a dynamic and premium feel.
- **📁 Sections**: 
  - Hero with High-Contrast CTA
  - Dynamic Stats Bar
  - Comprehensive Services Cards
  - Why Choose Us (Trust Indicators)
  - Before & After Gallery
  - Patient Testimonials
  - Integrated Booking Form
  - Professional Multi-column Footer

---

## 🛠 Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jodx19/Dental-Landing-page.git
   cd "dental Landing"
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
